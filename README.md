# blueprint

## 安装依赖
```
cnpm install
```

### 开发者模式
```bash
cnpm run serve
```

如果出现 `ERRONET`网络错误，需要配置 `hosts`文件。

> 在hosts文件中添加:
>
> 127.0.0.1 hostname(你的主机名)

### Compiles and minifies for production

```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
